# SiriLV Web IDE

Static VS Code-like browser IDE for GitHub Pages.

It does not run code and does not write directly to GitHub. It is designed as a portable editor that works on desktop and mobile:

- create files and folders
- rename, delete and duplicate files
- edit text files with tabs and line numbers
- autosave workspace in `localStorage`
- search across workspace
- Markdown and HTML preview
- import local files
- export workspace as JSON
- download current file
- dark/light theme
- mobile explorer drawer

Open: `https://sirilv.github.io/web/`
